Calidity Overview

The majority of home heating systems only react after a change in a system rather than proactively anticipating a change in a system (Eg: Boolean Thermostat). The goal here is to implement a smart home heating system that will record and monitor both the state and rate of change of internal and external temperatures & anticipate fluctuations in a closed system. We want to optimize cost and comfort. Emphasis will be on the monitoring and control software
