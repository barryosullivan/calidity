Calidity Dashboard

  1. Dashboard

Simple home heating/cooling systems only react after a change has occurred (e.g. Boolean Thermostat, Timer) rather than proactively anticipating temperature fluctuations in a system. If these external fluctuations could be anticipated – the internal fluctuations in temperature wouldn’t be as severe (if any at all). This intelligent home heating/cooling system can be used to record and monitor both the state and rate of change of internal/external temperature & weather data. The recorded data can then be used with simple machine learning to optimize cost and/or comfort. The simple & concise interface allows full control and monitoring over the entire system.
